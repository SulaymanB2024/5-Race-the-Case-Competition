# 5-Race-the-Case-Competition

Python financial-report generator and decision framework built for a Race-the-Case competition scenario.

The project generates forward-looking Beauty First Cosmetics 10-K-style reports at one-year, five-year, and ten-year intervals, then prints a strategic summary and recommendation framework.

Related portfolio work: https://sulayman-bowles.dev/work

## Repository Structure

- `main.py` and `Main.py` - entry points for the report generator
- `report_generator.py` - report-building logic
- `strategic_summary.py` - summary and recommendation output
- `decision_framework.py` - case decision framework
- `src/` and `bfc_reports.egg-info/` - supporting package files

## Run

```bash
python main.py
```

## Notes

This is a competition project archive. The generated reports are scenario analysis and should not be treated as audited financial reporting or investment advice.
